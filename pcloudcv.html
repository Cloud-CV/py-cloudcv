class pcloudcv.PCloudCV(file, list, login_required)[source]¶

Bases: threading.Thread

Creates a PCloudCV instance in a new thread.

Parameters:
  • file (str) – Path to config file.
  • list (dict) – Config input from user to override the Config file defaults.(Pass an empty dict to use defaults from config.json )
  • login_required (bool) – Specifies if a user wants to use a service that requires authentication. Default value is True.
